<Info>
  A whitelist por código só funciona se a whitelist for por perguntas, [clique aqui](/bots/tutoriais/whitelist/perguntas) para configurar a allowlist.
</Info>

## Como criar códigos de whitelist

1. Utilize o comando **/codiguin**.

<Frame>
  ![Codiguin Pn](/bots/images/whitelist/codiguin.png)
</Frame>

2. Selecione a opção **Adicionar Código**.

<Frame>
  ![Addcodiguin Pn](/bots/images/whitelist/addcodiguin.png)
</Frame>

3. Preencha com a quantidade de códigos que deseja criar, após preencher clique em **Enviar**.

<Frame>
  ![Addcodiguinmodal Pn](/bots/images/whitelist/addcodiguinmodal.png)
</Frame>

4. Pronto! Os códigos foram criados, lembrando que **cada código só pode ser usado uma vez**.
